### Capture stdout and stderr

From bash

```shell bash
echo "Hello from bash"

```


From default shell

```shell sh
echo "Hello from sh"

```

From default zsh

```shell sh
#!/usr/bin/env zsh
echo "Hello from zsh"

```
From default fish

```shell sh
#!/usr/bin/env fish
echo "Hello from fish"

```
