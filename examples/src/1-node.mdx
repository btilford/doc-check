### Setting up the NodeVMExecutor

This uses [VM2](https://github.com/patriksimek/vm2) to execute the scripts in a sandbox.
All the standard vm2 options can be passed in.

```typescript node
import {NodeVmExecutor} from '@btilford/md-check';

NodeVmExecutor.supply(); // Use defaults
NodeVmExecutor.supply({/* vm2 options */});
```
